## join.jsp 파일

value == null일때
공란이 비어있을 때는 알림창을 띄우고 해당 박스로 커서를 이동시킨다.

    공란의 빈공간 여부 확인 --> function checkValue() 
    알림창 --> alert("회원성명이 입력되지 않았습니다.");
    해당 버스로 커서 이동 --> data.custname.focus();
    공란이 비어있는지 확인 --> !document.data.custname.value

value == null이 아닐 때
공란이 채워져 있다.

return checkValue() == 공란 체크 함수
